In a HTML file, the <img> tag is used to insert images. The element is a void element, meaning it does not have a closing tag.

The <src> Attribute

The <img> tag requires the <src> attribute, which specifies the URL of the image to be displayed. The value of the <src> attribute can be a relative or absolute path URL. Different tools used to make and store websites will have different ways that image files can be stored and served, and this will affect the exact syntax required in the path provided to <src>.

The <alt> Attribute

The <alt> attribute is not required, but it is good practice to include it every time you use the <img> tag. The <alt> attribute provides alternative text for the image if it cannot be displayed. This is important for accessibility, as it allows screen readers to describe the image to users who are visually impaired.

Examples

The following code demonstrates how to use the <img> tag:

<img src="resources/image.png" alt="Description of the image">

This will render as:

Hover your mouse over the image to see the alternative text.

In the example below, a path is given to an image file that does not exist. Note how the browser displays the alternative text instead:

<img src="resources/nonexistent.png" alt="Description of the image">
