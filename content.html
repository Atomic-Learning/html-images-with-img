In an HTML file, the <img> tag is used to insert images. The element is a void element, meaning it does not have a closing tag.

The src Attribute

The <img> tag requires the src attribute, which specifies the URL of the image to be displayed. The value of the src attribute can be a relative or absolute URL. Different tools used to make and store websites will have different ways that image files can be stored and served, and this will affect the exact syntax required in the path provided to src.

The alt Attribute

The alt attribute is not required by the parser, but it should almost always be included when you use the <img> tag. The alt attribute provides alternative text for the image if it cannot be displayed. This is important for accessibility, as it allows screen readers to describe the image to users who are visually impaired. If an image is purely decorative, use an empty value (alt="") so assistive technologies can ignore it. Alternative text should describe the image's purpose in context, rather than generic text such as "image".

Examples

The following code demonstrates how to use the <img> tag:

<img src="resources/image.png" alt="Description of the image">
